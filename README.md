.　　**这是一个用C++实现的软光栅化渲染器, 纯粹用于练手, 它目前渣的不堪入目. 但是, 不排除将来可能加以完善并推广至全世界广泛使用的可能.**

.　　**用vs2017打开render.sln即可编译运行.**

.　　**bin目录下有已经编译好的可执行文件.**
